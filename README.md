# Analog-circuit
Record the process of analog circuit
## 第一章
### PN结
    人类电子行业的发展史是由电子管转向晶体管的过程，第一个晶体管的产生无疑是推动电子行业飞速发展的契机。
    本征半导体：没有参加杂质的如锗管和硅管
                由于其自身受到温度的影响会发生电子的移动
    杂质半导体：在本征半导体的基础之上掺加5价元素P 形成P型半导体
                在本征半导体的基础之上掺加3价元素B 形成N型半导体
    
    使用工艺将P型半导体和N型半导体进行拼接，即可在两端形成PN结
